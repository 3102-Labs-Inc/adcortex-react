Installation
============

To install the ADCortex Chat Client library, follow these steps:

1. **Install package**:
```sh
npm i adcortex-js
```

2. **Set Up Environment Variables**

Create a ``.env`` file in the project root and add your ADCORTEX_API_KEY:
```sh
ADCORTEX_API_KEY="your_api_key_here"
```
